# plausible-py

Python wrapper for Plausible API

Run pre-commit:

```
$ pre-commit install
$ pre-commit run --all-files
```

Run pytest:

```
$ pytest tests
```

Run coverage.py:

```
$ coverage run -m pytest tests
$ coverage report -m
$ coverage html
$ rm -R htmlcov  # once you've done check the report
```
